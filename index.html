网络重置

一键重置网络设置（Winsock/TCP-IP）、清理 DNS 缓存，帮助快速恢复网络连接。

操作说明

  1. 此操作将在本机执行 Winsock 重置、重设 TCP/IP 堆栈，并刷新 DNS 缓存。
  2. 请确保您以管理员身份运行本机上对应的操作（此页面仅提供命令与说明）。
  3. 执行前可尝试重新启动路由器或拔插网线以排查物理连接问题。
netsh winsock reset
netsh int ip reset
ipconfig /flushdns

提示：将以上命令在具有管理员权限的命令提示符或 PowerShell 中逐行执行。

检测与建议

  • 如果仍无法联网，请检查网卡驱动并尝试更新。
  • 确认没有代理或 VPN 影响网络路由。
  • 在企业环境下，联系网络管理员以确认 DHCP 与网关配置。

修改主机名 (macOS)

使用以下命令在 macOS 终端中快速修改计算机名称。

my="Example"; sudo scutil --set ComputerName "$my"; sudo scutil --set LocalHostName "$my"; sudo scutil --set HostName "$my"

提示：将 my="Example" 中的 Example 替换为新名称。

以管理员身份打开 PowerShell

以下为常见的在 Windows 上以管理员权限打开 PowerShell / Windows Terminal 的方法：

  1. 按 Win 键，输入 PowerShell，右键选择“以管理员身份运行”。
  2. 或 按 Win+X，选择“Windows 终端（管理员）”或“命令提示符（管理员）”。
  3. 在任务栏或开始菜单上找到 PowerShell，右键 -> 更多 -> 以管理员身份运行。

提示：打开后，将复制的命令粘贴并回车执行。若被系统提示确认，请选择“是”。
